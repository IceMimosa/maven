# Maven
Maven Repository

# Add repository

```xml
<repositories>
    <repository>
        <id>patamon.snapshot.repository</id>
        <name>github snapshot repository</name>
        <url>https://raw.github.com/icemimosa/maven/snapshot/</url>
    </repository>
    <repository>
        <id>patamon.release.repository</id>
        <name>github release repository</name>
        <url>https://raw.github.com/icemimosa/maven/release/</url>
    </repository>
</repositories>
```
